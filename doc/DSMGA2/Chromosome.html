    A Fixnum or a DSMGA2::Chromosome can be passed into. The former create a new Chromosome with length param. The latter #deep_clone the item.

def initialize(item)
  case item 
  when Fixnum
    @length = item
    @genes = Array.new(@length){yield}
    @value = fitness
  when DSMGA2::Chromosome
    deep_clone(item)
  end
end

def change(range, input_array)
  range.each do |i|
    @genes[i] = input_array[i-range.begin]
  end
  @value = fitness
end

def deep_clone(chromo)
  @length = chromo.length
  @genes = chromo.genes
  @value = chromo.value
end
    local_search() click to toggle source

    Bitwise flip and take if the fitness value increase, time-complexity is O(length)

    # File lib/dsmga2/chromosome.rb, line 33
def local_search
  (0...@genes.length).each do |i|
    tmp = self.class.new(self.itself)
    tmp.change((i..i), 1-@genes[i])
    deep_clone(tmp) if tmp.value > self.value
  end
end
